After flashing

Loading manifest.

  • Boot: press RESET after flashing if the board does not auto-reboot.
  • Channel: choose Stable or Dev before flashing. Dev fetches binaries live from the dev branch.
  • Provision: join Wi-Fi AP ESP32-UPS-Setup-XXXXXX, password configureme, then open http://192.168.4.1/.
  • Configure: log in as admin, then save Wi-Fi SSID/password, MQTT broker URL, optional MQTT user/password, optional device label, optional new config AP/admin password, and publish interval. Settings are stored in NVS.
  • Connect UPS: use ESP32-S3 USB OTG D-/D+ to the APC UPS USB port. Target APC VID 051D, Smart-UPS PID 0003.
  • Power path: keep Wi-Fi, MQTT broker, router, and JARVIS on UPS-backed power if you want outage alerts during outages.
